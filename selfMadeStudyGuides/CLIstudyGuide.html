CLI and how it applies

What is a CLI?

CLI stands for Command Line Interface. But what does that really mean? CLI's are used for a lot of the basic needs when developing software, including installation of software and configuration of how you want your computer to be set up. It can be used to move through your folders and documents, and even to create new folders and documents. You're even able to write information into documents from the CLI.

So that's all we need? A CLI?

While there are many things that a CLI can do on it's own, CLI's rely on things called package managers. Package managers are essential for installation and tracking of the software you have or want on your computer. It is kind of like a tool box that contains all of the tools and remembers which tools you have and helps you get new ones or remove old ones as you see fit.

So how do we use them?

CLI and package managers are super useful. Below you will see a list of common commands used with Linux:

Command What it does
ls list the directory system
cd Change directory (changes where you are in the file system)
cd .. Change directory 1 level up
mkdir make directory (create new folder)
rmdir remove directory (remove folder)
exit close the CLI window
